ColourLovers.com API Proxy

This proxy forwards any request to /api to http://colourlovers.com/api.

It is deployed on a single dyno heroku.

Example:

https://colorlovers.herokuapp.com/api/palettes/top would forward responses from
http://colorlovers.com/api/palettes/top
